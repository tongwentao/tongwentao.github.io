Privacy Policy for CF Analytics Pro Max

1. Information Collection and Use
CF Analytics Pro Max does not collect, store, or transmit any personal information to any third-party servers. All data processing is performed locally within the user's browser.

2. Data Source
The extension retrieves public submission data from the official Codeforces API (codeforces.com/api) based on the username displayed on the current profile page.

3. Permissions
The extension requires host permissions for Codeforces to provide data visualization features. It does not access any other sensitive browser data.

4. Changes to This Policy
This policy may be updated from time to time. Users are encouraged to check this page for any changes.

5. Contact
If you have any questions, please contact the developer via GitHub or email.
